# Suryansh Sharma — Portfolio

## 🚀 Deploy to GitHub Pages

### Step 1: Create a GitHub repo
1. Go to [github.com](https://github.com) → New Repository
2. Name it: `suryansh-portfolio` (or `<yourusername>.github.io` for root domain)
3. Set it to **Public**

### Step 2: Upload files
Upload **all files** from this folder to the repository root.

### Step 3: Enable GitHub Pages
1. Go to repo **Settings** → **Pages**
2. Under **Source**, select `main` branch and `/ (root)` folder
3. Click **Save**

### Step 4: Your portfolio is live! 🎉
URL: `https://<yourusername>.github.io/<repo-name>/`
